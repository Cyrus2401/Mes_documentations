### Commandes Git Memento
```bash
git status              # Voir l'état des fichiers
git add .               # Tout ajouter à l'index
git commit -m "Message" # Enregistrer les changements
git push origin main    # Envoyer vers le serveur (GitHub/GitLab)
git pull                # Récupérer les changements distants
git checkout -b feature # Créer et basculer sur une nouvelle branche
git checkout -d feature # Supprimer une branche
git checkout -D feature # Forcer la suppression d'une branche
git log --oneline       # Historique compact
```
